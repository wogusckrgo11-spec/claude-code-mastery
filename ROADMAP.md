# 개발자 웹 이력서 개발 로드맵

## 📋 프로젝트 개요
개발자 포트폴리오 역할을 하는 반응형 웹 이력서 개발 프로젝트입니다.
- **기술 스택**: HTML, CSS, JavaScript, Tailwind CSS
- **목표**: 전문적이고 모던한 개발자 웹 이력서 완성
- **타겟**: 채용담당자/면접관

## 💡 팁 및 주의사항

1. **Tailwind CSS CDN 사용 시**
   - 개발 단계에서는 CDN으로 빠르게 시작
   - 배포 전 빌드 프로세스 도입 권장

2. **이미지 관리**
   - 모든 이미지는 최적화하여 사용
   - 프로필 이미지는 정사각형 (300x300px 권장)
   - 프로젝트 썸네일은 16:9 비율 (600x337px 권장)

3. **접근성**
   - 모든 이미지에 alt 텍스트 추가
   - 색상 대비도 WCAG AA 기준 이상 유지
   - 키보드 네비게이션 지원

4. **SEO 최적화**
   - 의미있는 제목 구조 사용
   - Meta 태그 작성
   - Open Graph 태그 설정

5. **성능**
   - 불필요한 라이브러리 최소화
   - 이미지 지연 로딩 고려
   - CSS/JS 최소화
